Pokemon Mind Reader is a game I programmed in Java
It is essentially a rock paper scissor game but with "fire, water, grass". The program uses basic level mahcine learning to predict the users' future moves. The users' past moves are stored in a hashmap and used to calculate the frequency of move as well as the pattern of the most recent moves. Using this algorithm, the program attempts to beat the user at the game. I strength my knowledge in object oriented application development.
